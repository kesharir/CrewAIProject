## AI Agent
>AI Agent is a program that takes input, thinks and acts to complete a task using tools, memory and knowledge.
> Autonomous but narrow task specific and doesn't span multiple or evolving goals.


## Agentic AI
>Agentic AI is a system where one or more AI agents work autonomously, often over long tasks, making desisions using tools and even
> other agents to reach a goal
